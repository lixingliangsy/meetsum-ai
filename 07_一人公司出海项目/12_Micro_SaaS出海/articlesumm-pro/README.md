# 📰 ArticleSumm Pro

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/articlesumm-pro)

> AI-powered article summarizer for busy professionals. Save hours of reading time.

## ✨ Features

- ✅ **URL Summarization**: Paste a URL and get instant summary
- ✅ **Document Upload**: Summarize PDFs, DOCs, and text files
- ✅ **Custom Length**: Choose summary length (brief, detailed, bullet points)
- ✅ **Multi-Language**: Summarize articles in 30+ languages
- ✅ **Shareable Briefings**: Generate and share summary reports

## 🚀 How to Use

1. **Input Content**: Paste URL or upload document
2. **Customize**: Select summary length and format
3. **Get Summary**: Receive concise summary in seconds
