# 📸 Screenshot Annotator

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/screenshot-annotator)

> Capture, annotate, and share screenshots in seconds. Boost team communication.

## ✨ Features

- ✅ **Quick Capture**: Capture any part of your screen instantly
- ✅ **Annotation Tools**: Add arrows, text, shapes, and blur
- ✅ **Cloud Sync**: Sync screenshots across devices
- ✅ **Share Links**: Generate unique links for instant sharing
- ✅ **Export Options**: Save as PNG, JPG, or PDF

## 🚀 How to Use

1. **Capture**: Use keyboard shortcut to capture screen area
2. **Annotate**: Add annotations with intuitive toolkit
3. **Share**: Export or share via unique link
