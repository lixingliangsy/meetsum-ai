### 步骤 3 · 判断值不值得（需求强度分级）

**结论：需求强度 = 强。**

理由（均为公开可查信号）：
1. **结构性政策驱动，且每年复现**：2026 年《学位法》施行 + 多所高校将 AIGC 检测纳入毕业论文审核（知网/维普/万方/PaperPass 均上线 AIGC 检测）。这是强制性的、每年毕业季（5–6 月）复现的需求，非一时热点。来源：https://k.sina.com.cn/article_5953740931_162dee083067039tmq.html （纵览新闻/新浪，2026-05-22）；https://www.toutiao.com/a1865866191694848
2. **已验证的付费市场**：竞品定价集中在 **1–8 元/千字**（早标网 1.1、率零 2、去AIGC 3.5、嘎嘎降AI 4.8、比话 8），且普遍设免费额度做获客——说明用户愿意为「降AI率/降重」付费，市场已被教育。来源：https://www.zaobiao.net/blog/shi-ce-4-kuan-jiang-AIGC-lv-gong-ju-jiang-zhong-jiang-ai-de-ruan-jian-na-ge-hao-xue-sheng-dang-xing-jia-bi-shou-xuan-ju-ran-shi-ta-10792 ；https://devpress.csdn.net/v1/article/detail/159212406
3. **微信生态内原生使用 + 海量社媒讨论**：工具普遍提供「微信扫码/小程序/公众号」（用户在微信里直接用）；小红书近 6 个月「数百篇公开笔记、上万条评论」讨论降AI工具（今日头条 2026-07 汇总）；B站降AI 教程单条 5.9 万播放、合集 32.7 万播放。来源：https://www.toutiao.com/a7664585444075651619 ；https://space.bilibili.com/3546820059859151/

> 注意：需求「强」≠ 竞争「弱」。竞争同样强（红海）。本验证只回答「需求是否真实存在」→ **真实且强**；PurePen 的胜负手在差异化（语义保真+术语保护+合规护城河），由 07 转化闭环承接。

### 步骤 5 · 最小体验验证方案（零凭证怎么做）

> 设计层，发布属 D-08 人工动作，本报告不代发。

**方案 A · 公众号 + 智能体闭环（六步法标准形态）**
- 注册订阅号 `论文降AI率`（命名见步骤4）。
- **零凭证钩子**：菜单嵌「免费测AI率」入口，底层用 PurePen **已实跑的本地检测 heuristic**（detect.ts / risk.ts，纯前端/本地，不需 LLM 凭证）做 AI 率初诊，返回标红高危段报告。
- **最小体验**：用户粘贴段落 → 系统给 AI 率报告 + 高危段标红 → 引导「语义保真改写」预约/等待列表（不在此步实跑 LLM 改写，避免凭证依赖）。
- **观察**：菜单点击 / 对话发起 / 连续追问 / 关注 / 收藏分享。

**方案 B · 小红书钩子（不挂链，D-09 合规）**
- 发 1 篇以「论文降AI率」「降重降AI」为关键词的方法/避坑笔记（纯内容，**不挂任何外链/二维码/微信**）。
- 承接小红书站内搜索自然流量，观察私信咨询量（咨询再导到站外落地页，站外才放链接）。

**方案 C · 静态落地页钩子（零凭证）**
- 一个 HTML 落地页，title/meta 含「论文降AI率」「去AI痕迹」，部署到免费静态托管。
- 承接搜一搜/搜索曝光，观察访问与留资（邮箱/微信）。

> 三方案均可并行小流量测试；核心是「先看自然搜索流量与咨询，再决定是否投入凭证上线」。

### 步骤 6 · 转化指标定义（看转化，不看阅读量）

漏斗：**搜索词曝光（搜一搜/小红书搜索/搜狗微信） → 点击/访问（菜单/笔记/落地页） → 咨询（对话/私信） → 注册意向（关注/留邮箱/加微信）**

| 漏斗阶段 | 追踪指标 | 健康阈值（方向性） | 不合格线 |
|---|---|---|---|
| 曝光 | 搜索词自然曝光量（搜一搜/小红书搜索） | 持续有自然搜索进量 | 零曝光 |
| 点击 | 菜单/笔记/落地页点击率 | ≥ 行业内容均值 | 有曝光无点击 |
| 咨询 | 对话发起 / 私信咨询数 | 咨询率 > 0 且逐周上升 | 有点击无咨询 |
| 注册意向 | 关注 / 留邮箱 / 加微信 | 留资率 > 1%（小样本） | 无留资 |

**继续开发 vs 放弃 的数据决策门槛（参考六步法门槛「AI 数据≥人工 70% 且成本≤1/5 才值得做」，适配本验证）：**
- **继续**：核心搜索词有稳定自然曝光 + 咨询率>0 + 留资率>1% → 触发 H0 上线（配凭证部署）。
- **重估/放弃**：零曝光或零咨询持续 2–3 周 → 重估搜索词或暂停，不投入凭证。

## 3. 需求强度结论

**PurePen AI（净文）在微信生态的需求强度 = 强。**

三条核心理由：
1. **政策强制 + 年度复现**：2026《学位法》+ 高校 AIGC 检测纳入毕业审核，每年 5–6 月毕业季复现，属结构性刚需而非热点。
2. **付费市场已验证**：红海竞品定价 1–8 元/千字且设免费获客额度，用户付费意愿已被教育。
3. **微信原生使用 + 海量讨论**：工具普遍提供微信扫码/小程序/公众号；小红书数百篇笔记上万评论、B站降AI 教程单条 5.9 万播放、主流媒体专题报道——需求真实性「强」。

> 但需要清醒：**竞争同样强（红海）**。PurePen 的切入点不是「再做一次降AI率」，而是以「语义保真（SPS>0.7）+ 术语保护 + 合规去AI味」建立差异化信任锚，把媒体批评的「灰色产业链」反转为合规护城河（对齐 D-11）。

## 5. 风险与下一步

**风险**
- **R1 红海竞争**：降AI率/降重赛道竞品数十款、价格透明，PurePen 须靠「语义保真+术语保护+合规」差异化突围，否则易陷价格战。
- **R2 合规舆论风险**：媒体已批评「降AI率灰色产业链」（大河报/央视新闻）。PurePen 必须坚守 D-11（不代写、不承诺清零），把「合规去AI味」做成护城河，而非蹭灰色需求。
- **R3 季节波动**：需求随毕业季（5–6 月）峰值、7 月后骤降；需用自媒体/职场副利基（去AI味）平滑淡季，并提前蓄量。
- **R4 信号近似局限**：微信指数需登录未直连，本报告用公开近似；正式上线后应以真实搜一搜曝光/菜单点击数据校准。

**下一步（衔接）**
1. **主理人人工动作（D-08）**：择一方案（建议 A 公众号+智能体）注册订阅号 `论文降AI率`、搭建零凭证钩子，观察自然流量与咨询。
2. **H0 上线闸门**：若验证通过（见步骤6 门槛），补齐 Supabase/LLM/Waffo 凭证 + Vercel 部署，进入真实服务与收费（对齐 `mvp-plan.md` 的 H0→H1/H4/H5）。
3. **进入 07 转化闭环**：承接命名/钩子词，设计完整获客路径与定价 A/B（D-19 框架），不与本验证层越界。
