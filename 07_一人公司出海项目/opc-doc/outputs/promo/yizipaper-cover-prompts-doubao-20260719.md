# 一字成文 · 小红书封面图提示词包（豆包专用 · 4 张）

> 主理人汇编 · 2026-07-19 · 对应 `yizipaper-xhs-notes-extra-20260719.md` 与 `yizipaper-promo-pack-20260719.md` 的封面建议
> 为什么用豆包而不是 ImageGen：小红书封面是"超大中文标题 + 排版"型，通用图像模型对中文渲染极不稳定（乱码/缺笔画/错位）；豆包对中文更友好，更适合出带准确文字的成品封面。
> 尺寸统一：**竖版 3:4（1080×1440）**，小红书信息流标准比例。

---

## 关于"调用 skills 生成图片"的说明

- 我有图像生成能力（`ImageGen` 工具，约 5–10 credit/张），但它是**通用底模**，**不擅长渲染中文大字标题**，直接出"带准确文字的封面"风险高（文字易乱码）。
- 因此推荐路线：
  1. **成品封面（带标题文字）** → 用 **豆包** 跑下面提示词（对中文支持最好）。
  2. **无文字背景底图**（学术氛围 / 科技纹理）→ 若你需要，我可用 `ImageGen` 生成几张纯视觉底图，你再在 Canva / 稿定 / 醒图 上叠文字（需先确认，会消耗 credit）。
- 若你坚持让我直接用 `ImageGen` 出带文字封面，我也可以试，但**需你接受文字可能不准、且消耗 credit** 的前提。

## 通用技巧（提高豆包出图准确率）

1. **文字指令必加**：每段提示词末尾都带了"请准确渲染以上中文字符，不要变形或替换为乱码"——这能显著降低乱码率。
2. **一次出多张**：豆包可一次生成 2–4 张，挑最清晰的一张；若标题仍不准，单独重跑该张并加"仅生成背景，文字我后续叠加"。
3. **兜底方案**：若豆包文字始终不准，改用「Canva / 稿定设计 / 醒图」的"小红书封面"模板，把上面的标题文字手动输入——模板字体可控、零乱码。
4. **配图一致性**：四篇笔记封面建议统一圆角卡片 + 同款描边风格，矩阵感更强、账号更专业。
