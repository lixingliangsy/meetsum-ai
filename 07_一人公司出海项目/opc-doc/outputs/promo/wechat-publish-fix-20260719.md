# 公众号发布实操修正（v2）· 2026-07-19

> 目的：修正此前 wechat-ready 文件中「正文 `<a href>` 超链接」在公众号后台会失效的问题，让 4 篇推文**真正可直接发文**。

## 一、结论：公众号可以直接发文，它是设计好的"承接层"
- 小红书的限制（不挂链 / 不二维码 / 不 AI 托管发布）**只针对小红书平台**，与公众号无关。
- 公众号（订阅号 / 服务号）是站外转化承接页，正是放转化信息的地方，**可直接发文、无"不能发"限制**。
- 唯一实操坑：公众号**正文不能插"可点击的外链"**（绝大多数账号正文超链接会被微信过滤，点了不跳）。解决见第二节。

## 二、公众号外链限制与解法
| 方式 | 能否跳外链 | 用法 |
|---|---|---|
| 正文 `<a href>` / 富文本超链接 | ❌ 会被过滤 | 不要用 |
| 正文**明文 URL**（用户复制） | ✅ 可复制 | 放 yizipaper 链接明文 |
| 正文插入**二维码图片** | ✅ 可扫 | 扫码直达（图片不被过滤） |
| 底部**「阅读原文」按钮** | ✅ 可跳外链 | 填 yizipaper 链接 |

## 三、统一 CTA 块（替换原 HTML 里的 `<a href>`，4 篇推文通用）
直接复制下面这段粘到每篇推文结尾（用排版工具排好即可，链接是明文不是超链接）：

```
🔗 一字成文官网直达：
https://www.yizipaper.com/?tuid=189990
（复制上面链接到浏览器打开，或点文章底部「阅读原文」也能直达）

🎁 专属福利码：LYG646（注册时填入，享专属权益）

⚠️ 本人体验分享，非官方账号；内容由 AI 辅助生成，
请遵守学术规范，将其作为辅助工具使用，勿直接提交未校验内容。
```

> 备选：若你已有**稳定落地页 URL**（见第四节 Vercel），可把"官网直达"换成落地页 URL，落地页内按钮再跳 yizipaper（双跳更稳、且落地页不被平台限制）。当前 CloudStudio 临时链接不建议长期放推文。

## 四、公众号发文步骤（电脑端，修正版）
1. 登录 mp.weixin.qq.com → 内容与互动 → 图文消息 → 新建。
2. 填标题 / 封面图 / 摘要（见各篇 `yizipaper-wechat-ready*.md` 的标题与结构）。
3. 正文：把 wechat-ready 的 Markdown 经**秀米 / mdnice** 排版后粘贴；把原 `<a href>` 段**替换为上面的「统一 CTA 块」**（明文 URL，不依赖超链接）。
4. （可选）用草料二维码 cli.im 把 `https://www.yizipaper.com/?tuid=189990` 生成二维码图片，上传插入正文。
5. 编辑页底部**「原文链接」框**：填 `https://www.yizipaper.com/?tuid=189990`（唯一可点跳转的外链入口）。
6. 不勾「声明原创」（推广文非原创）；定时群发选 **20:00–22:00** 黄金时段（按 `yizipaper-publish-schedule-20260719.md` 日期）。
7. 预览到手机自查：链接 / 福利码 / 免责是否齐全。
8. 群发。
