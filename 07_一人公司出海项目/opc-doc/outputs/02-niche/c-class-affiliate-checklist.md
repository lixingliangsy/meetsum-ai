# C 类选品 + 挂链操作清单（小红书买手合作 / 笔记带货）

> 主理人汇编 · 基于 2026-07-16 联网核查（小红书《交易导流违规管理细则》、图书网络发行治理公告、笔记带货开通规则、选品中心佣金口径）。
> 用途：服务养号期现金流补充（见 `content-nurturing-strategy.md` §2 C 类）。属市场研究输入，非成员正式产出。

---

## 0. 一句话总原则
C 类 = 你**不生产、不发货**，只靠"笔记挂链赚佣金"。**唯一合规路径 = 小红书站内闭环（商品卡/选品中心）**；**任何站外链接/二维码/微信号 = 违规封号红线**。

---

## 1. 读书笔记挂书链接赚佣金 —— 合规吗？

### 结论：✅ 合规，但有严格前提
- **可以**：在小红书**站内**通过"笔记带货 / 商品卡 / 选品中心"挂载**小红书商城内的正版书**商品链接，用户站内购买，你拿佣金。图书本身是合规商品（非盗版/非法出版物）。
- **不合规（踩雷即罚）**：
  1. 笔记/图片/评论/私信放**淘宝/京东/拼多多等站外书链接、口令、二维码** → 违反《交易导流违规管理细则》，可删内容+限流→封号，最高 2 万违约金。
  2. 卖/分享**盗版电子书、高清影印书** → 2026-04-08 起 5 平台（腾讯/抖音/淘天/拼多多/小红书）联合专项治理重点，侵权冻结。
  3. 用"一元一本""书店倒闭"等虚假低价引流 → 专项治理。
  4. 错放类目逃避审查。
- 你作为**买手/达人分销**，不是售书主体，资质（出版物经营许可证等）在供应商侧，你只需：挂正版、不盗版、不虚假宣传、站内闭环。

### 关键澄清（修正早前简报精度）
- 早前说"0 粉可开橱窗/笔记带货"大体成立，精度补充如下：
  - **0 粉可开**：个人薯店、站内「商品卡/选品中心」分销权限（挂小红书商城内商品）。
  - **通常需 ≥1000 粉解锁**：第三方商品挂载（淘宝/京东跳转）、"买手合作/好物推荐"深度权限、蒲公英品牌商单。
  - 你 96 天店、粉丝可能不足 1000 时：**先走 0 粉可开的站内商品卡分销**；涨到 1000 粉再扩第三方/商单。
  - 每条笔记**最多挂 3 个商品**；商品必须与内容强相关（否则"虚假种草"限流）。

## 2. 详细方案：读书笔记带书佣金 落地 6 步

### 步骤 1 · 开通权限（0 粉可做）
- 路径：我 → 左上三横 → 创作中心 → 更多功能 → **商品卡** → 立即开通（实名认证 + 绑定收款账户，0 粉无门槛，即时开通）。
- 备选：创作中心 → 好物推荐/买手合作（若提示需粉丝，先涨粉再开）。

### 步骤 2 · 选品（图书垂直）
- 进「选品中心」→ 搜"图书"类目 → 筛选：销量高、佣金合理、正版（小红书商城内书均经平台审核）。
- **契合主利基**：优先选**职场成长/效率/简历求职类书**（《金字塔原理》《高效能人士的七个习惯》《你的降落伞是什么颜色》《简历写作》等），内容复用简历/PPT 流量。
- **佣金预期**：图书类佣金约 **5–10%**（书利润薄，低于美妆 10–30%、个护）。单本定价 30–60 元 → 单笔佣金约 1.5–6 元。

### 步骤 3 · 内容（读书笔记种草）
- 结构：痛点/场景 → 书中方法 → 我的实践 → 自然引出书（商品卡放中后段）。
- 保持"分享语感"，不硬广；禁"最/第一"等极限词；封面用实拍/书影，不出现站外信息。

### 步骤 4 · 挂链
- 编辑笔记 → 添加商品（组件）→ 从「我的选品」勾书 → 确认封面价格 → 发布。
- 每篇 ≤3 商品；建议单篇 1 本主推 + 评论区引导"点笔记商品卡看同款"。

### 步骤 5 · 数据追踪
- 创作中心 → 数据中心 → 电商数据：商品点击率 / 转化率 / 收益金额。
- 优化：转化高的书加更；点击低检查内容匹配度。

### 步骤 6 · 合规自查（每条发布前）
- 无站外链接/二维码/微信；商品与内容相关；非盗版；无虚假低价；商单标"广告"。

## 3. 推进合作商家的方式

| 路径 | 门槛 | 做法 | 说明 |
|---|---|---|---|
| **A 选品中心自发挂链** | **0 粉** | 选品中心选书 → 挂商品卡 → 平台结算佣金 | **推荐起步**。你无需谈商家，小红书已对接入驻供应商（含出版社/图书公司） |
| **B 蒲公英品牌商单** | **≥1000 粉** | 入驻蒲公英 → 出版社/图书公司下单定制笔记 | 必须走官方报备 + 标注"广告/合作"，否则违规；报价约粉丝数 0.1–0.3 元/条 |
| **C 主动推进** | ≥1000 粉 | 站内通过蒲公英/官方图书活动对接；谨慎用站内私信联系出版社账号 | 话术避免导流/硬广；**绝不**接"挂我淘宝书链接给你佣金"私单（站外导流违规） |

> 图书商家自身需出版物经营许可证（供应商侧资质），你作为分销不涉及。

## 4. 风险护栏（红线清单）

- 🚫 任何笔记/图片/评论/私信/主页/包裹：禁微信/手机号/QQ/二维码/淘宝京东拼多多链接/口令/水印。
- 🚫 禁盗版电子书、影印书、非法出版物。
- 🚫 禁"一元一本""书店倒闭"等虚假低价引流。
- 🚫 商品与内容不相关（虚假种草）。
- 🚫 商单不标"广告"。
- **处罚阶梯**：删内容+3 天限流 → 封禁 30 天+扣信用分 → 永久封号/关商品分享功能/最高 2 万违约金。

---

## 5. 与整体养号策略的衔接

- **养号期（0→1000 粉）**：用**路径 A（0 粉商品卡）**挂职场成长类书，既涨粉（读书内容高收藏）又赚薄佣；内容主力仍放 A 类（简历/PPT）涨粉。
- **1000 粉后**：扩路径 B 蒲公英商单 + 第三方商品；同时自营虚拟品资质达标即上架。
- **现金流排序**：自营虚拟品（资质后）> 高佣 C 类（养生/个护 10–30%）> 图书佣金（5–10%，作契合补充，非主现金流）。
