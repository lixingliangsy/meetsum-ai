## 0. 策略总纲与平台规则基线

### 0.1 两阶段框架
```
阶段一 · 养号期（0 → 1000 粉，约 30–60 天，全程在 96 天内）
   只做高质量种草笔记，建立「职场新人简历+汇报」标签，不碰店铺成交
   └─ 变现过渡：1000 粉前即可用「买手合作 / 笔记带货挂他人商品」0 粉起步赚佣金
阶段二 · 带货期（≥1000 粉 + 资质临近 180 天）
   开通笔记带货 / 直播带货，挂载自营虚拟品（资质达标后）+ 买手合作实物佣金
   内容从「纯种草」转向「种草 + 转化」混合
```

### 0.2 2026 笔记带货最新规则（关键，决定变现节奏）
- **2026-01 新规**：**0 粉即可开橱窗 + 笔记带货**（挂载商品赚佣金）；**≥1000 粉可开直播带货**（此前需 1000 粉才能开笔记带货，现已放开）。
- 单篇笔记最多挂载 **18 个商品**；评论区挂商品链接**不降权**（平台鼓励）。
- 佣金率因类目而异（美妆 10–30% 等），实物类目走「买手合作」链路，你无需生产、无需 180 天资质即可赚佣金。
- **自营虚拟商品上架仍受 180 天 + 30 笔记 + 1000 粉 + 创作证明 约束** → 这是你（96 天店）当前唯一卡住的环节，故店铺成交必须后置到资质达标。

> 含义：养号涨粉的 30–60 天里，**现金流可以靠「带别人家的实物赚佣金」先跑起来**，不必干等 180 天；等资质一达标，立刻切自营虚拟品放量。

## 3. 从养号到带货的内容过渡策略

### 阶段一 · 养号期（0 → 1000 粉）
- **内容定位**：纯价值种草，建立「职场新人简历+汇报」标签，不挂链、不硬广、不露微信/二维码。
- **内容矩阵比例**（参考 40/30/20/10）：
  - 40% 干货价值（简历怎么写 / 述职 PPT 怎么搭 / 周报模板）
  - 30% 人设故事（应届生踩坑 / 转行经历 / 职场成长）
  - 20% 热点借势（毕业季 / 金三银四 / 年终述职季）
  - 10% 互动引流（投票 / 提问 / 「你想要哪套模板」）
- **涨粉打法**：前 10 篇定赛道标签；黄金定位公式「自我擅长（模板/AI）+ 用户需求（求职职场）+ 竞争空白（组合+AI 指南）」；长尾词 SEO 卡位（如「应届生简历模板免费」「第一次述职 PPT」）。
- **现金流过渡**：1000 粉前即可开「买手合作 / 笔记带货」挂 C 类实物赚佣金（0 粉门槛），不依赖 180 天资质。

### 过渡期（800 → 1000 粉）
- 开始埋钩子：「想要这套模板的姐妹评论区扣 1」「主页有领取方式」（平台内允许引导，去二维码/微信号）。
- 私域沉淀：用高风险类目（AI 提示词 / Notion / 笔记模板）作加微赠品钩子，但站内话术合规。
- 测试挂链：试挂 C 类实物佣金链接，观察 CES 与转化，验证内容-商品匹配度。

### 阶段二 · 带货期（≥1000 粉 + 资质临近）
- 开通笔记带货 / 直播带货，内容从「纯种草」转向「种草 + 转化」混合（约 60% 种草 + 40% 转化）。
- 挂载优先级：先 C 类实物佣金（已验证）打底现金流 → 资质一达标立刻上架 A 类自营虚拟品（简历钩子 + 汇报 PPT 主利润）→ B 类蓝海虚拟（AI 头像壁纸）试水。
- 升单路径：笔记引流 → 私信/主页 → 网盘/私域 → Excel 看板等高客单升单（资质达标后）。

### 过渡关键动作清单
1. 养号前 10 篇：锁定「简历 + 汇报 PPT」标签，统一封面风格（复用已有 8 张封面图）。
2. 第 11–30 篇：稳定每周 2–3 篇，混入 1–2 篇 C 类带佣内容测试佣金转化。
3. 800 粉起：每篇末尾加合规钩子话术，观察私信/评论引流效果。
4. 1000 粉：正式开笔记带货，挂 C 类实物；同步备齐 A 类创作证明，等 180 天资质一达标即切自营。

## 4. 笔记发布节奏建议（基于 2026 算法）

### 4.1 算法要点（决定节奏逻辑）
- **三层权重**：基础权重（账号健康/历史）/ 动态权重（近期活跃度）/ 实时权重（发布后 3 小时互动）。
- **CES 互动评分** = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8（评论与关注权重最高 → 引导留言、做关注钩子最有效）。
- **内容质量分占 40%**：纯 AI 无二次创作水文献降权 30% 甚至封禁 → 你的模板需加「AI 优化指南/SOP」二次创作，既合规又提权。
- **搜索流量占比 >35%**：每篇埋 2–3 个长尾关键词（如「应届生简历模板」「第一次述职 PPT」），吃搜索流量。
- **黄金 3 小时互动窗口**：发布后 3 小时内互动密度决定实时权重 → 必须卡点发布 + 主动回复评论。
- **断更惩罚**：断更 >7 天动态权重 **-30%**，>15 天判低活跃 → **绝对不能断更**。

### 4.2 发布频率
- **新号启动（前 30 天）**：目标 20–30 篇；前 10 篇集中定标签；**每天 1–2 篇，坚持 21 天**建立活跃基线。
- **稳定期**：每周 **≥2–3 篇**保底，节假日/毕业季/述职季加更。
- **铁律**：宁可少更，不可断更；断更 7 天以上权重掉 30%，前功尽弃。

### 4.3 四大黄金发布时段（卡点发布 + 守 3 小时互动）
| 时段 | 时间窗 | 适合内容 |
|---|---|---|
| 早间 | 6:30–9:00 | 通勤碎片阅读（干货/清单类） |
| 午间 | 11:30–13:30 | 午休刷屏（轻松/测评类） |
| 傍晚 | 17:30–19:30 | 下班放松（人设/故事类） |
