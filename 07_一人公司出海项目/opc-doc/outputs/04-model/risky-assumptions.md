## 假设清单（按验证优先级）

### H0 · 上线前置三缺口假设【P0 · 模型成立前提】
- **陈述**：Supabase 迁移 SQL 执行 → LLM/Waffo/Embedding 凭证配置（.env 去 REPLACE_ME）→ Vercel 部署三步完成后，PurePen 才能从"工程就绪"转为"可服务可收费"；同时站外流量层（落地页/affiliate/内容）须启动，否则无用户。
- **为何最危险**：01 资源明确当前闭环"工程就绪但未上线未起量"——Supabase 迁移待执行、凭证未配、Vercel 待部署、流量层未启动。四者任一卡住，订阅收入无从谈起，整盘不成立。
- **验证方式**：执行迁移 SQL + 配齐凭证 + 部署；监测落地页/affiliate 曝光与首单。
- **证伪后果**：凭证/部署卡住 → 维持工程态无收入；流量层起不来 → 产品再好无用户，须重估 GTM 组合。

### H1 · 保真溢价接受度假设【P1 · 价值验证】（继承 03-H1）
- **陈述**：用户愿为「语义保真（SPS>0.7）+ 术语保护 + 合规去AI味」相对纯降率黑产支付**溢价**（即"保专业"值多少钱）。
- **为何危险**：UVP 与 Unfair Advantage 全建立在此；若用户只肯为低价降率买单，差异化坍塌回价格战。注意：**保真"能力"已由 P2 实跑收敛**，本假设仅剩"溢价付费意愿"待验。
- **验证方式**：定价梯度小样本 A/B（保真订阅 vs 纯降率低价）测价格接受度与转化差。
- **证伪后果**：溢价不成立 → 退守低价走量或强化保真感知价值（如视频演示 SPS 度量）。

### H2 · 细分扩展可迁移性假设【P1 · 扩展经济性】（继承 03-H2）
- **陈述**：备选② 非母语学术（向上细分）与 备选③ 企业泛化（泛化扩展）的**付费意愿与可迁移性**足以支撑扩展产品线（升单客单更高）。
- **为何危险**：若扩展群付费弱或迁移成本高，则主利基跑通后无法靠升单放大 LTV，增长天花板低。
- **验证方式**：主利基跑通后，测非母语/企业场景包转化与客单。
- **证伪后果**：扩展弱 → 收束到学术主利基单线，放弃②③升单。

### H3 · 合规护城河假设【P1 · 留存/复购】（继承 03-H3）
- **陈述**：2025-09《标识办法》后，"主动合规去AI味"能实际转化为**差异化信任与复购**，而非纯成本项。
- **为何危险**：若用户不 care 合规、只比价格，则合规叙事无溢价、纯防御，护城河失效。
- **验证方式**：07 转化闭环中的留存 / 口碑 / 复购数据。
- **证伪后果**：合规不转化为信任 → 降为基线要求，靠保真能力竞争。

### H4 · 站外获客冷启动假设【P1 · 核心打法】（替代旧 H2 私域转化）
- **陈述**：站外获客主轴（落地页 v3 / affiliate / 小红书种草标识 AI 不挂链）能低成本引量到独立站/私域并完成订阅转化，绕开 XHS 虚拟商品门槛。
- **为何危险**：六维"可达=4"依赖站外获客，但**人脉/内容/流量层三圈规模待核实**（01 待核实项），冷启动能否跑通未验证；若获客成本过高，单位经济崩。
- **验证方式**：落地页/affiliate 曝光 → 独立站/订阅漏斗转化；小红书种草涨粉不挂链引流实测。
- **证伪后果**：获客成本过高或起量失败 → 重设渠道组合或引入人脉/内容杠杆（待核实项）。
- **依赖 01 待核实**：人脉（学术/开发者/AI 工具三圈规模）影响冷启动与口碑扩散；内容（学术成果/技术博客/案例库）影响信任背书与内容获客成本。

### H5 · 订阅单位经济 / 价格锚假设【P1 · 收入结构】（结合 01 待核实）
- **陈述**：订阅制（钩子检测报告免费/低价 → 主力改写订阅 → 升单非母语·企业）能被学术主利基接受，且价格锚高于纯降率黑产、区别于 QuillBot/Wordtune（$8.33/$13.99）中文脱节定价；同时 ARPU 足以覆盖 LLM 处理成本（PRD KPI：<30% ARPU）。
- **为何危险**：学生预算有限（付费=4）；若订阅 ARPU 低于 LLM 成本，单位经济不成立，规模越大越亏。
- **验证方式**：06 MVP 小样本订阅转化 + 07 真实成交数据回测定锚；单位经济测算（ARPU vs LLM 成本）。
- **证伪后果**：订阅 ARPU 不足 → 调价或改一次性按量付费结构。
- **依赖 01 待核实**：资金（初始投入规模未披露 → 决定冷启动期能否补贴/获客投入）；人脉（种子用户/口碑节点）；内容（案例库背书降低获客成本）。
