- **P0** = MVP 必须，阻塞上线

```
M0(基础设施) ──► M1(接入解析) ──► M2(降重引擎) ──┬─► M3(AI检测) ──┐

所有模块 ──► M7(运营/合规/风控)                  GTM 管线并行不阻塞
```
