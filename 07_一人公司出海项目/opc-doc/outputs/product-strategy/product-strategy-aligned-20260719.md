## §1 产品定位 PRD（对齐 02 / 03 / 04）

> 本 § 将 `prd-zh/PRD.md`（方向② 单产品 PRD）锚定到 02 利基 + 03 价值主张 + 04 商模，消除其中残留的"四方向"语境，统一为 PurePen 单一主产品视角；功能清单对接 m0.2 / m0.3 已实跑能力。

### 1.1 一句话定位（引 03 主推版 · 不重拟）

> **为需过知网 / 维普 / 朱雀 AI 率红线的学术与专业写作者，提供「语义保真（SPS>0.7）+ 术语保护 + 合规去AI味」的中文降重降AI SaaS——不只把 AI 率压下去，更把你的专业表达、专有名词和学术诚信原样保住。**

### 1.2 目标用户（对齐 02 主利基）

中文语境下的**学术 / 专业写作者**——硕博研究生、高校教师、科研人员、专业撰稿人（毕业论文、核心刊投稿、基金本子），需过知网/维普/朱雀 AI 率红线、且要求专业表达不被破坏。

| 子人群 | 最痛场景 | 入口产品 |
|---|---|---|
| A · 硕博 / 高校师生（最高频主入口） | 自己用 AI 辅助写，AI 率过高过不了，手动改又破坏专业度，盲审/答辩 deadline 紧 | 检测报告 + 语义保真改写 |
| B · 科研人员 / 专业撰稿人 | 术语密集、专有名词多，通用改写把核心概念改丢，需经得起同行审阅 | 术语保护 + 改写 |
| C · 非母语学术写作者（备选②向上细分，H2 留待主利基跑通后） | 原创被高误判（61.3%），申诉无据 | 非母语降误判场景包 |

### 1.3 核心问题（对齐 04 Problem）

1. **P1 · AI 率红线刚性且大**：本科 ≤20%、硕士 ≤15%、核心刊 ≤10%；过不了不能投、不能交、不能盲审/答辩。
2. **P2 · 通用改写/翻译工具破坏专业表达**：QuillBot（~$8.33/月）、Wordtune（~$13.99/月）面向英文通用改写，中文学术语境水土不服、贵、与中文检测生态脱节。
3. **P3 · 纯降率黑产暴力洗稿**：为压 AI 率把语义改得面目全非、术语丢失，甚至承诺"绕过检测/不标注过审"，踩学术诚信与合规双红线。
- 伴生：P4 检测器口径不一（6 检测器一致性仅 ~63%）、P5 术语被改坏、P6 合规收紧后去AI味两难（非母语误判率 61.3%）。

### 1.4 核心功能流程（对齐 03 Value Map + 04 Solution + 已实跑能力）

```
上传文档 → ① 多检测器 AI 率报告（诚实标注：6 检测器一致性~63% / 非母语误判率61.3% / 软降级不造假）
        → ② 语义保真改写（SPS = embedding 余弦 > 0.7，2026 新指标，与 P2 阈值对齐）
        → ③ 术语保护（专有名词必含校验，零丢失）
        → ④ 合规去AI味（依 2025-09 标识办法，不绕过检测、不承诺 100% 过检）
        → 订阅（钩子免费报告 → 主力改写订阅 → 升单非母语/企业）
```

## §3 调研对齐（对齐 06 / 07 联网发现）

> 将基线 `market-research-2026-07.md` 的联网发现，对接 02 利基 / 03 价值 / 06 MVP，统一为 PurePen 单一视角。

### 3.1 检测器脆弱性 → 诚实定位 + 软降级不造假（对齐 03 Gains / 06 §2.1 / 07 合规护栏）

- 6 个主流检测器一致性仅约 **63%**；非母语写作者误判率高达 **61.3%**；改写后检测准确率骤降。
- **产品应对（已实跑）**：多检测器报告钩子**诚实标注误判风险**；检测引擎未配置即返回"未度量"绝不编造（软降级，`src/lib/detect.ts`）。
- **定位含义**：不承诺 100% 过检，把"诚实、可控、可辩护"做成差异化信任锚，而非黑产式虚假承诺。

### 3.2 SPS>0.7 新指标（对齐 02 边界变动 / 03 G1 / P2 工程）

- 语义保真（SPS = 全文 embedding 余弦）>0.7 判高保真，是 2026 中文降重降AI 新指标。
- **与 PurePen P2 阈值对齐**：`SEMANTIC_FIDELITY_THRESHOLD=0.7` 已实跑收敛（01 资源确认），构成"保真优先"的可量化支点，区别于纯降率暴力洗稿。

### 3.3 2025-09-01《标识办法》合规去AI味 = 护城河（对齐 02 边界 / 03 H3 / 04 UVP）

- 标识办法施行后，主动合规去AI味从成本转为护城河；黑产（合规脆弱）与大厂（不敢碰/检测方只检不改）均难复制。
- **产品应对**：不绕过检测、不承诺 100% 过检；合规去AI味内建为价值主张与风控红线（07 §7）。

### 3.4 国际对标（QuillBot ~$8.33/月、Wordtune ~$13.99/月）→ 不照搬、保真溢价差异化（对齐 03 §3.1 / 04 pricing）

- 二者面向**英文通用改写**，中文学术语境水土不服、贵、与中文检测生态（知网/维普/朱雀）脱节。
- **PurePen 定价逻辑（04 pricing-notes）**：不为"通用改写"计价，为"中文保真降AI（SPS>0.7 + 术语保护 + 检测口径适配）"计价；避价格战、避黑产低价；具体数字 H1 小样本 + H5 回测后定稿（本阶段不写）。

### 3.5 7 月毕业季窗口不可逆（挂 H4，对齐 09 瓶颈）

- 当前处于毕业季尾声 + 考研准备期，论文降重/降AI 需求年度高峰；等凭证到了再起号，旺季流量红利即过。
- **结论（09 复盘）**：现在不是"等产品上线"，而是"上线前先把站外内容/流量资产跑起来"（M1 / Quick Win A），这是唯一可掌控且不可逆的瓶颈突破口。

### 6.2 合规红线（单列 · 零容忍，与 07 §7 / 09 §6 一致）

> 违规 = 0 是 H0 前提硬约束（封号/封站即清零蓄水）。依 2025-09-01《标识办法》，主动合规去AI味为护城河。

- 🚫 **小红书站内（笔记/图片/评论/私信/主页）禁**：二维码、微信号、外链、`?tuid` 等导流标识；福利码仅纯文字灰色呈现（07 路径 B/C 口径，绕开虚拟商品门槛）。
- 🚫 **不设"一键降AI""包过检测"等虚假/极限承诺**；标题/正文禁极限词。
- 🚫 **主动合规去AI味**：多检测器报告诚实标注（6 检测器一致性 ~63%、非母语误判率 61.3%、软降级不造假）；护城河靠"合规 + 保真 + 术语保护"而非黑产。
- 🚫 **不启用无人值守 / AI 托管式自动发布**（D-08 半自动姿态：属明确违规运营=封禁），内容发布保留人工目检。
- ⚠️ **独立站订阅成交需 H0（部署+凭证+流量启动）打通 + Waffo 收款配置后方可收真实费**（凭证未配不标"可收费"）。
- 🚫 **不代写、只做生产引擎**（用户自用，D-11）；降重须保原创性、反对学术不端。
