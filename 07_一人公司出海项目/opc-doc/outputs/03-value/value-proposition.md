## 1. 客户细分再确认（基于 02 阶段主利基 + 备选）

**主利基人群**：中文语境下的学术 / 专业写作者——硕博研究生、高校教师、科研人员、专业写作者（如行业报告 / 智库撰稿人）。
**连续场景**：写作成稿 ➜ 过 AI 率检测（知网 / 维普 / 朱雀）→ 投稿 / 提交 / 盲审 / 答辩。
**核心诉求**：必须压到单位 / 期刊要求的 AI 率红线以下，**但专业表达、术语、论证逻辑不能被破坏**。

**细分再拆（3 个子人群，共用同一价值主张骨架）**：
- **A. 硕博 / 高校师生（最高频主入口）**：最痛"自己用 AI 辅助写了，检测 AI 率过高过不了，手动改又破坏专业度"，盲审 / 答辩 deadline 紧。
- **B. 科研人员 / 专业撰稿人**：最痛"术语密集、专有名词多，通用改写把核心概念改丢"，且需经得起同行审阅。
- **C. 非母语学术写作者（备选②向上细分）**：最痛"原创却被高误判为 AI（误判率高达 61.3%）"，申诉无据、最不公平。

> 三类子人群在"成稿 → 过检 → 提交"链路上的位置不同，但**痛点性质一致**（要降 AI 率、又不能破坏专业度、还要合规），故可用同一套"语义保真 + 术语保护 + 合规去AI味"组合承接，差异仅在场景包的垂直颗粒度（如"硕博论文降AI包""科研术语保护包""非母语降误判包"）。

## 2. 价值主张画布

### 2.1 Customer Profile（用户侧）

#### Customer Jobs（核心任务，6 条）
1. **过 AI 率红线投稿 / 提交**：论文、报告、稿件需通过知网、维普、朱雀等 AI 检测，降到单位 / 期刊要求的 AI 率阈值以下。
2. **保住专业表达与学术严谨**：改写后术语、公式、论证逻辑、专业语气不漂移，能经得起导师 / 同行 / 审稿人审阅。
3. **守住专有名词与引用规范**：机构名、理论名、方法名、专业缩写必含且准确，参考文献与标注合规。
4. **在 deadline 前快速降 AI 率**：投稿截止、盲审、答辩前的改稿窗口短，需快速产出可用版本。
5. **应对多平台检测口径不一**：同一文本在不同检测器结论不同，需要稳定可控的改写策略。
6. **降低"被误判 AI 写作"的合规与声誉风险**：非母语写作者误判率高达 61.3%，需可解释、可辩护的成稿。

#### Pains（痛点，6 条）
1. **通用改写 / 翻译工具破坏专业表达**：QuillBot（~$8.33/月）、Wordtune（~$13.99/月）等面向英文通用改写，中文学术语境水土不服、贵、非专门，且与中国检测生态（知网 / 维普 / 朱雀）脱节。
2. **纯降率黑产"暴力洗稿"**：为压 AI 率把语义改得面目全非、术语丢失，甚至承诺"绕过检测 / 不标注过审"，踩学术诚信与合规双红线。
3. **检测器口径不一、不可控**：6 个主流检测器一致性仅约 63%，改完此处过、彼处不过，焦虑且反复返工（改写后检测准确率亦骤降）。
4. **非母语写作者被高误判**：误判率高达 61.3%，辛苦写的原创被当成 AI，申诉无据、最不公平。
5. **术语 / 专有名词被改坏**：核心名词被替换或丢失，专业度崩塌，导师 / 审稿人一眼识破。
6. **合规收紧后的"去 AI 味"两难**：2025-09-01《人工智能生成合成内容标识办法》施行，既要降 AI 率又要主动合规标注，黑产方案直接违规。

#### Gains（收益期望，6 条）
1. **拿到语义保真的降 AI 版本**：SPS（全文 embedding 余弦）>0.7 即高保真，专业含义不漂移（2026 新指标，与 PurePen P2 阈值对齐）。
2. **专有名词零丢失**：术语保护机制确保机构 / 理论 / 方法名必含且准确。
3. **诚实可解释的降率结果**：不承诺 100% 过检（与 PurePen 软降级原则一致），但给出可辩护、可复现的改写策略与依据。
4. **一处改写多平台稳健**：针对知网 / 维普 / 朱雀主流口径统一优化，减少因 63% 检测器一致性带来的反复返工。
5. **合规去 AI 味即护城河**：主动合规而非黑产绕检，长期积累信任与口碑。
6. **非母语写作者被公平对待**：降误判、给出原创性佐证，缓解 61.3% 高误判带来的不公。

### 2.2 Value Map（供给侧）

#### Products & Services（你的组合）
- **PurePen AI 降重降AI SaaS（中文语义保真改写引擎）**：核心交付，P2 阈值 SPS≥0.7 高保真改写，区别于纯降率暴力洗稿。
- **术语保护模块**：专有名词必含校验，机构 / 理论 / 方法名零丢失。

#### Pain Relievers（如何缓解痛点，对应 Pains）

- **PR2 → P2（黑产暴力洗稿）**：不做暴力洗稿，明确"不绕过检测、不承诺 100% 过检"，语义保真优先，在合规与学术诚信护栏内降率。

- **PR4 → P4（非母语高误判）**：术语保护 + 非母语专门降误判策略，缓解 61.3% 高误判。
- **PR5 → P5（术语被改坏）**：专有名词必含校验，术语零丢失。
- **PR6 → P6（合规两难）**：2025-09 标识办法后主动合规去AI味，把"合规"从成本转为护城河。

#### Gain Creators（如何创造收益，对应 Gains）

- **GC2 → G2（术语零丢失）**：术语保护机制确保专有名词必含准确。
- **GC3 → G3（诚实降率）**：不承诺 100% 过检，给可辩护、可复现的改写依据（软降级原则）。

- **GC5 → G5（合规护城河）**：主动合规去AI味，长期积累信任。

## 3. 价值主张陈述（主推版）

> 必含要素自检：✅ 中文语境学术 / 专业写作者降重降AI ｜ ✅ 语义保真（SPS>0.7）+ 术语保护 + 合规去AI味差异化 ｜ ✅ 区别于纯降率黑产 / 国际通用改写工具

### 一句话版
**为需过知网 / 维普 / 朱雀 AI 率红线的学术与专业写作者，提供「语义保真（SPS>0.7）+ 术语保护 + 合规去AI味」的中文降重降AI SaaS——不只把 AI 率压下去，更把你的专业表达、专有名词和学术诚信原样保住。**

### 展开版
针对中文语境硕博 / 高校教师 / 科研人员 / 专业写作者在"成稿 → 过检 → 提交"链路上的连续需求，PurePen 提供以「中文语义保真改写引擎」为核心、并叠加「术语保护 + 合规去AI味 + 多检测器口径适配」的专业降重降AI SaaS。

与纯降率黑产（暴力洗稿、承诺绕过检测）不同，PurePen **不绕过检测、不承诺 100% 过检**（与软降级原则一致），而是以 SPS（全文 embedding 余弦）>0.7 为高保真阈值，在保住专业含义的前提下压低 AI 率；与 QuillBot（~$8.33/月）、Wordtune（~$13.99/月）等国际通用改写工具不同，PurePen 专为中文学术语境训练、针对知网 / 维普 / 朱雀主流口径优化、内置术语保护，不与中国检测生态脱节。

由此形成"保真优先 + 术语保护 + 合规护栏"的复合差异化——在 2025-09《标识办法》施行后，主动合规去AI味从成本转为护城河；而 6 检测器仅 63% 一致性、非母语误判率 61.3% 的现实，恰恰证明"诚实、可控、可辩护"的降率策略才是用户真正需要的。
